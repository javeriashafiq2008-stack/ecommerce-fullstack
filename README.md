## Features
* **Global State Management:** Integrated React Context API (`ShopContext`) for global data flow and cart persistence.
* **Dynamic Routing:** Implemented client-side routing via React Router with dynamic parameters for product discovery.
* **Responsive UI/UX:** Built a mobile-first, clean interface using Tailwind CSS.
* **Authentication Modules:** UI ready for User Registration, Login, and Profile Management.

## Tech Stack
* **Frontend:** React.js, Tailwind CSS, Vite
* **State Management:** React Context API
* **Backend:** Node.js, Express.js (In Progress)
* **Database:** MySQL (In Progress)
